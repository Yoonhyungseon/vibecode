# 숫자 맞추기 게임 (Guessing Game)

간단한 콘솔 기반 숫자 맞추기 게임입니다. 1부터 100 사이의 랜덤 숫자를 맞추면 승리합니다.

웹(UI) 버전이 추가되었습니다: `web/index.html` 파일을 브라우저에서 열면 깔끔한 웹 인터페이스로 게임을 플레이할 수 있습니다.

웹 버전 실행 방법:

- 로컬에서 파일을 직접 열기: 파일을 파일 관리자(또는 브라우저에서) `web/index.html`을 열면 됩니다.
- 로컬 HTTP 서버로 서빙하기 (안전한 방법): 프로젝트 루트에서 아래 명령 실행 후 브라우저로 http://localhost:8000/web 를 엽니다.

```bash
python3 -m http.server 8000
```

웹 버전 기능 요약:

- 깔끔한 UI + 애니메이션
- 정답을 맞추면 페이지 중앙에 고양이가 춤추는 애니메이션이 약 10초 동안 재생됩니다 (원할 경우 즉시 닫기 가능)
- 축하 시 Confetti 효과

사용법

```bash
python3 guessing_game.py
```

명령어
- 숫자 입력: 1에서 100 사이의 정수 입력
- q 또는 Ctrl+C: 게임 종료

힌트
- 입력한 숫자가 정답보다 작으면 "더 큽니다"
- 입력한 숫자가 정답보다 크면 "더 작습니다"
